# hhplus-spring
항해플러스 사전 스터디 - Spring

# 개요
항해플러스 본 과정 진행 전 간단한게 SpringBoot에 대해 학습하고자 한다.

# 목표
- REST API 명세 작성하기
- ERD 작성하기
- 체계적으로 프로젝트 일정 관리하기
- CRUD 및 회원가입/로그인 기능 구현
